1. 产品定位与愿景

产品名称：Yoru 夜次元（或：NeonDive 霓虹潜行）

Slogan：越过边界，遇见同类。（Cross the line, find your kind.）

产品定位：严格限制18岁以上用户下载与使用的二次元潮流文化社区。

核心愿景：为成年 ACG 爱好者打造一个没有低龄粉圈争吵、没有过度内容自我阉割的“成年人避风港”。在这里可以自由探讨硬核游戏、Galgame、深夜番剧、青年漫画、微恐悬疑以及高质量的同人创作。

2. 视觉与 UI/UX 设计风格（年轻化）

主色调（暗黑赛博/酸性设计）：默认全局深色模式（Dark Mode），背景采用深邃的黑灰色（如 #121212）。点缀色使用高饱和度的霓虹粉、电光蓝、酸性绿，营造赛博朋克与迷幻感。

UI 元素：

毛玻璃效果（Glassmorphism）：大量使用半透明磨砂玻璃卡片，增加 iOS 平台的原生高级感。

流体动画与微交互：点赞特效为像素爆裂或霓虹闪烁，页面切换采用丝滑的阻尼弹簧动画。

全屏沉浸感：抛弃传统的带有明显边界的 Tab bar，采用悬浮式底部导航栏（类似灵动岛的胶囊设计）。

字体：使用无衬线黑体（如苹方）搭配部分极具张力的机甲风/像素风英文字体作为标题。

3. 核心功能模块划分

🔑 模块一：绝对结界（18+ 严格准入机制）

为了符合 iOS App Store 审核（设定为 17+）并真正实现 18+ 社区，必须建立极具仪式感且严格的门禁。

入站仪式：App 启动是一段简短的神秘视觉动画，提示“前方属于成人领域”。

身份验证：

基础门槛：注册需绑定实名手机号。

硬核解锁（可选）：通过“成人向 ACG 知识问答”（如某部深夜番剧的制作公司、某款限制级硬核游戏的梗）+ 活体人脸识别（仅验证年龄）来解锁核心“深夜区”（NSFW/R18+ 讨论版块）。

📱 模块二：世界线（双信息流广场）

表世界（DAY）：白天展示潮流二次元内容。如：主机/PC游戏评测、手办盲盒开箱、高质量 Cosplay、漫展集锦、潮玩谷子交易。

里世界（NIGHT）：每天晚上 20:00 自动解锁（或下拉切换）。专属成年人的信息流，内容包括：

青年向/R18+ 漫画探讨

猎奇/恐怖类 ACG 作品推荐

情感树洞与深夜扩列（成年人视角的社交）

“防折叠”的高清插画（画师发布的带有 NSFW 标签的作品，默认模糊，点击需指纹/FaceID确认后显示）。

🪐 模块三：地下城（硬核兴趣圈子）

抛弃传统的“贴吧”模式，采用类似 Discord 的服务器/频道结构。

频道分类：

[Galgame品鉴室]、[硬核机甲整备班]、[痛车/潮牌联名]、[同人本资源交换（合规前提下）]、[Vtuber 深度切片]。

门槛机制：部分“深水区”频道需要等级达到，或者由内部用户发放“邀请码”才能进入，保证圈子的纯粹性。

🎭 模块四：假面舞会（匿名与匹配社交）

成年人的社交需要一定的安全距离，抛弃即时语音带来的社交压力，主打异步与灵魂共鸣。

虚拟分身：用户不使用真实头像，系统提供高自由度的 Live2D 捏脸系统或赛博朋克风格的头像生成器。

“共鸣”匹配：类似 Tinder 的右滑匹配，但卡片展示的不是自拍，而是用户的“ACG 成分表”（Steam 游戏库、看过的番剧清单、XP 偏好标签）。

电波信箱（异步匿名交流）：放弃连麦功能，采用类似“赛博漂流瓶/笔友”的模式。用户可以写下一段深夜感悟、XP 探讨或冷门番剧推荐，封装成“电波”发送。收到电波的人可以选择回复，双方通过异步信件（文字+图片）交流，保持社交的舒适与神秘感。

4. 商业化与变现模式

黑卡会员（Yoru Black）：提供动态专属头像框、更改 App 图标颜色、发帖自定义渐变色昵称、解锁无限次“共鸣”匹配。

数字资产与装扮：与知名画师、虚拟主播（Vtuber）联名，推出限定版的主题皮肤、聊天气泡、虚拟礼物。

潮玩/同人电商接入：内置“黑市”版块，用于用户之间交易绝版手办、本子、痛包、谷子等，平台抽取极低比例的手续费。

5. iOS App Store 审核与安全合规策略

18+ 社区在 Apple 的审核政策下面临一定挑战，需要巧妙平衡：

App 评级：必须在 App Store 后台将内容评级设定为 17+（频繁/强烈的成人/性暗示题材、卡通暴力等）。

NSFW 过滤器：App 默认状态必须屏蔽严重违规内容（如直接露点的色情图片）。可以采用 NSFW 标签 + 严重模糊滤镜，用户需主动在设置中开启“展示敏感内容”并进行额外的年龄确认。

完善的举报与拉黑机制：必须提供用户拉黑、帖子举报功能（UGC 审核三要素），以应对苹果的审核要求。# youyue
